# NuitDuCode

Un jeu de tir spatial développé en python avec pygame surtout pour tricher a NuitDuCode.

## Structure du projet

```
nuitducode/
├── src/
│   └── nuitducode/           # package principal
│       ├── entities/         # entités du jeu
│       ├── core/             # logique du jeu
│       ├── ui/               # interface utilisateur
│       └── utils/            # utilitaires
├── assets/                   # images, sons, fonts
├── tests/                    # tests unitaires
├── main.py                   # point d'entrée
└── requirements.txt          # dépendances
```

## Installation

```bash
pip install -r requirements.txt
```

## Lancement

```bash
python main.py
```

## Développement

Le projet suit une structure de package Python moderne avec séparation des responsabilités.
