Un jeu de tir spatial développé en python avec pygame pour NuitDuCode ce qui te permettra d'une belle petite base que tu pourras travailler ou la rendre directement comme ça

## Stack

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![Pygame](https://img.shields.io/badge/Pygame-3776AB?style=for-the-badge&logo=python&logoColor=white)
![NuitDuCode](https://img.shields.io/badge/NuitDuCode-000000?style=for-the-badge&logoColor=white)

## Structure du projet

```
nuitducode/
├── src/
│   └── nuitducode/
│       ├── entities/         # joueur, ennemis, projectiles
│       ├── core/             # boucle de jeu et logique principale
│       ├── ui/               # menus, HUD, écrans
│       └── utils/            # fonctions utilitaires partagées
├── assets/                   # images, sons, fonts
├── tests/                    # tests unitaires
├── main.py                   # point d'entrée
└── requirements.txt          # dépendances
```

## Installation

```bash
pip install -r requirements.txt
```

## Lancement

```bash
python main.py
```
